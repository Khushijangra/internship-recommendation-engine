# 🚀 SIH 2025 — Internship Recommendation Engine (PS: 25034)

### ✅ Objective
Create a mobile-ready, simple, AI-powered recommendation engine for internship matching, as per PM Internship Scheme.

### 💡 Features
- User login/registration
- Profile form: education, skills, sector, location
- ML-based recommendation engine
- Outputs 3–5 internship cards
- Mobile & regional language friendly UI

### 🧠 AI Approach
- TF-IDF + Cosine Similarity
- Rule-based fallback matching
- ML-light as required by problem statement

### 🛠️ Tech Stack
- Frontend: React.js + Tailwind
- Backend: FastAPI + Firebase Auth
- ML: scikit-learn (TF-IDF)
- Database: PostgreSQL
- Deployment: Railway + Vercel
